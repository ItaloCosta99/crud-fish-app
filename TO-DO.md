Desafio – Projeto LinkFish – Seleção de Bolsistas

Desenvolver uma tela para realizar o CRUD de uma Cliente. Nessa tela EU devo cadastrar, alterar, excluir e visualizar os clientes cadastrados;
Para isso utilize seus conhecimentos em:
•	ReactJS;
•	Material-UI (https://mui.com/pt/);
•	Axios;
•	Redux;
•	Entre outros;
Obs: Os dados podem ficar salvos temporariamente, ou seja, a tempo de execução, utilizando um Array;
OU
Pode utilizar uma API para isso. 

Atributos: (nome da empresa, nome do responsável; cidade; estado; logradouro; Numero; CEP; CPF/cnpj; porte; telefone; email)
*Requisitos: 
•	Porte: Pequeno, Médio, Grande
•	Cpf/cnpj -> Deve aceitar os 2 valores; 
•	Todos os campos devem possuir formatação.
